# cashSystem

> 一个基于vue.js构建的一个收银管理系统，主要是为了练习vue技术。

### 技术栈

vue + webpack + vueRouter + elementUi + mock数据

### 项目启动

```
# 安装依赖
npm install

# 项目启动
npm run dev

# 项目打包
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
